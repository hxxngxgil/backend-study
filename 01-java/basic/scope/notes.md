# Scope (스코프)

## 스코프란
스코프(scope)는 변수에 접근할 수 있는 범위를 의미한다.  
변수는 선언된 위치에 따라 사용할 수 있는 범위가 결정된다.

---

## 지역 변수(Local Variable)
지역 변수는 특정 블록 {} 안에서 선언된 변수이다.  
해당 블록 내부에서만 사용할 수 있으며 블록을 벗어나면 사용할 수 없다.

예시

int a = 10;

if (a > 5) {
    int b = 20;
    System.out.println(b);
}

b는 if문 내부에서 선언되었기 때문에 if문 밖에서는 사용할 수 없다.

---

## 스코프 규칙
변수는 자신이 선언된 블록 내부에서만 사용할 수 있다.

예시

if (true) {
    int x = 10;
}

System.out.println(x);  
→ 오류 발생

---

## 스코프 존재 이유

### 1. 변수 충돌 방지
같은 이름의 변수를 서로 다른 블록에서 사용할 수 있다.

예시

for (int i = 0; i < 5; i++) {
}

for (int i = 0; i < 5; i++) {
}

---

### 2. 메모리 효율
변수는 스코프가 끝나면 메모리에서 제거된다.

---

### 3. 코드 가독성 향상
변수의 사용 범위를 제한하여 코드 이해가 쉬워진다.
