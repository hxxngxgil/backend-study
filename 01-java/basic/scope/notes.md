# Scope (스코프)

## 스코프란
스코프(scope)는 변수에 접근할 수 있는 범위를 의미한다.  
변수는 선언된 위치에 따라 사용할 수 있는 범위가 결정된다.

---

## 지역 변수(Local Variable)
지역 변수는 특정 블록 {} 안에서 선언된 변수이다.  
해당 블록 내부에서만 사용할 수 있으며 블록을 벗어나면 사용할 수 없다.

예시

int a = 10;

if (a > 5) {
    int b = 20;
    System.out.println(b);
}

b는 if문 내부에서 선언되었기 때문에 if문 밖에서는 사용할 수 없다.

---

## 스코프 규칙
변수는 자신이 선언된 블록 내부에서만 사용할 수 있다.

예시

if (true) {
    int x = 10;
}

System.out.println(x);  
→ 오류 발생

---

## 스코프 존재 이유

### 1. 변수 충돌 방지
같은 이름의 변수를 서로 다른 블록에서 사용할 수 있다.

예시

for (int i = 0; i < 5; i++) {
}

for (int i = 0; i < 5; i++) {
}

---

### 2. 메모리 효율
변수는 스코프가 끝나면 메모리에서 제거된다.

---

### 3. 코드 가독성 향상
변수의 사용 범위를 제한하여 코드 이해가 쉬워진다.

---

# Type Casting (형변환)

형변환은 한 데이터 타입을 다른 데이터 타입으로 변환하는 것이다.

---

## 자동 형변환 (Implicit Casting)

작은 범위의 타입에서 큰 범위의 타입으로 변환될 때는  
자동으로 형변환이 이루어진다.

예시

int a = 10;
double b = a;

이 경우 int 값이 자동으로 double 타입으로 변환된다.

---

## 명시적 형변환 (Explicit Casting)

큰 범위의 타입을 작은 범위의 타입으로 변환할 때는  
개발자가 직접 형변환을 해줘야 한다.

예시

double a = 10.5;
int b = (int) a;

결과

b = 10

소수점 부분은 버려진다.

---

## 계산과 형변환

### 1. 같은 타입끼리 계산

같은 타입끼리 계산하면 결과도 해당 타입으로 계산된다.

예시

int a = 10;
int b = 20;

int result = a + b;

결과 타입은 int이다.

---

### 2. 다른 타입끼리 계산

서로 다른 타입이 계산되면  
더 범위가 큰 타입으로 자동 형변환 후 계산된다.

예시

int a = 10;
double b = 20.5;

double result = a + b;

이 경우 int 값이 자동으로 double로 변환된 후 계산된다.

따라서 결과 타입도 double이 된다.

---

## 꼭 기억할 것

1. 같은 타입끼리 계산하면 해당 타입으로 계산된다.
2. 다른 타입끼리 계산하면 더 큰 범위의 타입으로 자동 형변환된다.

예시

double result = 10 + 20.5

→ int 값이 double로 자동 변환된다.
