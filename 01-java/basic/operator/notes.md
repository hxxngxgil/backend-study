# Operator (연산자)

## 연산자란
연산자는 변수나 값에 대해 계산을 수행하는 기호이다.

---

# 산술 연산자

기본적인 수학 연산을 수행하는 연산자이다.

예시

int a = 10 + 5;
int b = 10 - 5;
int c = 10 * 5;
int d = 10 / 5;
int e = 10 % 3;

---

# 문자열 더하기 연산자

Java에서는 문자열(String)과 `+` 연산자를 사용하면 문자열이 연결된다.

예시

String result = "Hello " + "Java";

결과

Hello Java

문자열과 숫자를 더하면 숫자가 문자열로 변환된다.

예시

String result = "Age : " + 20;

결과

Age : 20

---

# 연산자 우선순위

연산자는 계산할 때 **우선순위**가 존재한다.

기본적인 우선순위

1. 괄호 ()  
2. 단항 연산자 (++, --, !, ~, new, (type))  
3. 산술 연산자 (*, /, % 우선, 그 다음 +, -)
4. Shift 연산자 (<<, >>, >>>)
5. 비교 연산자 (<, <=, >, >=, instanceof)
6. 등식 연산자 (==, !=)
7. 비트 연산자 (&, ^, |)
8. 논리 연산자 (&&, ||)
9. 삼항 연산자 (? :) 
10. 대입 연산자 (=, +=, -=, *=, /=, %= 등등)

예시

int result = 10 + 2 * 3;

계산 순서

2 * 3 = 6  
10 + 6 = 16

---

# 증감 연산자

값을 **1 증가 또는 감소**시키는 연산자이다.


++  값 1 증가 
--  값 1 감소 

예시

int a = 10;
a++;

결과

a = 11

---

## 전위 증감

연산이 먼저 수행된다.

예시

int a = 10;
++a;

결과

a = 11

---

## 후위 증감

값을 먼저 사용한 후 증가한다.

예시

int a = 10;
a++;

결과

a = 11

---

# 비교 연산자

두 값을 비교하여 **true 또는 false**를 반환한다.

==  같다 
!=  다르다 
>   크다 
<   작다 
>=  크거나 같다 
<=  작거나 같다 

예시

int a = 10;
int b = 20;

boolean result = a < b;

결과

true

---

# 논리 연산자

여러 조건을 결합할 때 사용한다.

&&  AND 
||  OR 
!   NOT 

예시

int age = 20;

boolean result = age > 18 && age < 30;

---

# 대입 연산자

값을 변수에 저장할 때 사용한다.

=   대입 
+=  더해서 대입 
-=  빼서 대입 
*=  곱해서 대입 
/=  나눠서 대입 

예시

int a = 10;

a += 5;

결과

a = 15
