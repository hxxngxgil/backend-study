# Array (배열)

## 배열이란
배열은 같은 타입의 여러 데이터를 하나의 변수에 저장할 수 있는 자료구조이다.

예를 들어 여러 개의 변수를 따로 선언하는 대신 배열을 사용하면  
데이터를 효율적으로 관리할 수 있다.

---

## 배열 선언과 생성

배열을 사용하려면 먼저 배열을 선언하고 생성해야 한다.

예시

int[] numbers = new int[5];

설명

int[] → 배열 타입  
numbers → 배열 변수 이름  
new int[5] → 크기가 5인 배열 생성

---

## 배열 사용

배열은 인덱스를 사용하여 값을 저장하거나 조회할 수 있다.

인덱스는 0부터 시작한다.

예시

int[] numbers = new int[3];

numbers[0] = 10;
numbers[1] = 20;
numbers[2] = 30;

---

## 배열 리팩토링

배열을 사용하면 여러 개의 변수를 하나로 관리할 수 있어 코드가 간결해진다.

예시

변수 여러 개 사용

int student1 = 90;
int student2 = 80;
int student3 = 70;

배열 사용

int[] students = {90, 80, 70};

---

## 2차원 배열

2차원 배열은 배열 안에 배열이 들어있는 구조이다.

예시

int[][] arr = new int[2][3];

설명

2 → 행(row)  
3 → 열(column)

---

## 2차원 배열 리팩토링

2차원 배열도 초기값을 사용하여 간단하게 생성할 수 있다.

예시

int[][] arr = {
    {1, 2, 3},
    {4, 5, 6}
};

---

## 향상된 for문 (for-each)

배열이나 컬렉션을 반복할 때 사용하는 반복문이다.

일반 for문보다 코드가 간결해진다.

예시

int[] numbers = {10, 20, 30};

for (int number : numbers) {
    System.out.println(number);
}
