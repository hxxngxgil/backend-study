# Method (메서드)

## 메서드란
메서드는 특정 기능을 수행하는 코드의 묶음이다.  
코드를 재사용하고 구조를 나누기 위해 사용한다.

---

## 메서드 정의

메서드는 반환 타입, 메서드 이름, 매개변수로 구성된다.

예시

public static int add(int a, int b) {
    return a + b;
}

---

## 메서드 사용 (호출)

정의된 메서드는 호출하여 사용할 수 있다.

예시

int result = add(10, 20);

---

## 반환 타입

메서드는 실행 결과를 반환할 수 있다.

- 반환값이 있는 경우 → 타입 명시
- 반환값이 없는 경우 → void 사용

예시

void print() {
    System.out.println("출력");
}

int sum(int a, int b) {
    return a + b;
}

---

## 메서드 호출과 값 전달

Java에서는 메서드 호출 시 **값이 전달된다.**

즉, 변수 자체가 아니라 **값이 복사되어 전달된다.**

예시

int a = 10;

change(a);

→ a 값만 복사되어 전달됨

---

## 중요 개념

Java는 변수를 복사하는 것이 아니라  
**변수에 담긴 값을 복사해서 전달한다.**

---

## 메서드 형변환

메서드 호출 시 매개변수 타입이 다르면  
자동 형변환 또는 명시적 형변환이 발생할 수 있다.

예시

double result = add(10, 20);

→ int 값이 double로 변환될 수 있음

---

## 메서드 오버로딩

같은 이름의 메서드를 여러 개 정의할 수 있다.  
단, 매개변수의 타입이나 개수가 달라야 한다.

예시

int add(int a, int b) {
    return a + b;
}

double add(double a, double b) {
    return a + b;
}

---

## 메서드의 장점

- 코드 재사용
- 코드 가독성 향상
- 모듈화 (기능별 분리)
- 유지보수 용이
- 확장성 증가
- 추상화 가능
- 테스트와 디버깅 용이
